 # Kurulum & İçerik

- İlk olarak bilgisayarına [Node JS](https://nodejs.org/en/) indir.

- Daha sonra altyapıyı indirip ./beş_config.js dosyasını doldur.

	- Bu kadar bot hazır

	-  `<Prefix>kimlik:` Komuduyla Kimlik Oluşturabilirsiniz.
